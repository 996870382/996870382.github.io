html笔记

盒模型

  • 内容区:定义的高度和宽度
  • 边框，边框粗细 边框颜色 边框类型
  • 内边距，内容区与边框之间的距离
  • 外边距，元素与元素之间的距离

盒模型的真实大小固定不变，增加内边距，改变的是内容区
外边距，在水平方向上使用auto可以水平居中，但是垂直方向上不可以和元素所占的空间位置相关

浮动

  • 浮动的元素性质发生了改变，变成了内联块元素
  • 浮动元素脱离了文档流
  • 浮动的元素会导致父元素的高度塌陷
  • 浮动的元素在遇到包含框或者另外一个浮动元素，则停止浮动

高度塌陷解决:开启bfc

  • overflow:hidden; 副作用相对来说较小，但是会剪切掉超出元素的其他部分
  • float:left; 也会导致父元素脱离文档流
  • position:absolute;也会导致父元素脱离文档流
  • display:inline-block;但是会导致宽度与内容宽度一致

定位:指定元素在浏览器中任意摆放位置

默认值:没有开启定位 static

  • 相对定位:相对于元素原来的位置发生偏移量，不会脱离文档流，元素性质也不会发生改变
  • 绝对定位:脱离文档流，元素性质也发生改变，元素层级上升一个层级
    1. 如果祖先元素开启定位，则绝对定位的元素相对于开启定位的最近的祖先元素发生偏移量
    2. 祖先元素没有开启定位，则以浏览器作为参照物发生偏移量
  • 固定定位:(是绝对定义的一种特殊情况)，参照物永远是相对于浏览器窗口而定位，不会随着浏览器的滚动发生改变

弹性盒模型：包含了弹性容器和弹性子项目

  1. 开启弹性容器 display:flex;
  2. 决定子元素在主轴的排列方向 flex-direction:column;
    • row 水平方向
    • column 垂直方向
  3. 弹性子元素水平方向上的对齐方式 justify-content:flex-start
    • flex-start 左对齐
    • flex-end 右对齐
    • center 居中对齐
    • space-between 两端对齐
    • space-around 均匀对齐
  4. 强制换行 flex-wrap:wrap
  5. 在垂直方向上，子元素之间的对齐方式 justify-content:flex-start
    • flex-start 顶部对齐
    • center 居中
    • flex-end 底部对齐

背景图片 background-image:url(img/banner1.jpg)

  1. 背景平铺方式 background-rrepeat:repeat-x
    • no-repeat 不平铺
    • repeat-x 水平方向平铺
    • repeat-y 垂直方向平铺
  2. 背景图像的大小 background-size:cover
  • 100% 水平方向和元素宽度一样，垂直方向自动等比例缩放
  • 100% 100% 水平方向和垂直方向都是100%
  • cover 等比例平铺，图像等比例放大到容器的大小，多余的不显示

网站的三要素

  • 网站的标题 <title>我的网站</title>
  • 网站的关键词 <meta name="keywords" content="前端 css html" />
  • 网站描述信息 <meta name="description" content="这是网站的描述">
  • 网站标题的小图标 <link rel="icon" href="图片url" />

图片

  1. opacity:0.5 图片透明度(0~1)
    • -moz-opacity:0.5 兼容火狐的图片透明度
    • -webkit-opacity:0.5 兼容谷歌的图片透明度
  2. 高、宽：只设置一个值，高宽等比例缩放
  3. 图片的阴影 box-shadow: 2px 2px 20px 5px black;
    参数列表:
    - 阴影的水平偏移量，大于零向右偏移，小于零向左
- 阴影的垂直偏移量，大于零向下，小于零向上
- 阴影的模糊程度，值越大越模糊，不可为负值
- 阴影颜色

块级元素

  • 没有设置高度和宽度的情况下，宽度为100%，高度为内容区域的高度，自动适应高度
  • 独占一行
  • 可以设置高度和宽度

内联元素

  • 没有设置高宽的情况下，高宽都为内容区域的高宽
  • 内敛元素水平方向上排列
  • 不可以设置高宽
  • 内边距水平方向上正常，垂直方向上没有实际更改元素所占空间，并且会让元素错位

内联块元素 display:inline-block;

  • 符合内联元素中的水平排列
  • 符合块级元素的可以设置高宽
  • 符合内联元素中的，没有设置高宽的情况下，高宽和内容区域保持一致
  • 符合块级元素的内边距情况
  • 在垂直方向上，两个内联块元素以底部对齐

元素的位置改变

  • 绕中心点旋转 transform:rotate(5deg)
  • 平移
    • 水平方向上平移 transform:translateX(5px)
    • 垂直方向上平移 transform:translateY(5px)
  • 放大
    • 以中心点放大 transform:scale(5)
    • 水平方向放大 transform:scaleX(5)
    • 垂直方向放大 transform:scaleY(5)
  • 改变元素的中心点 transform-origin:left top

透视距离，给父元素添加透视距离，作用于子元素，可以看到3d效果，则子元素设置3d设置 perspective:1000px

动画

  1. 创建动画

    @keyframes,定义关键帧
 */
 @keyframes go{
     0%{
         opacity: 0.5;
         margin-top: 0px;
     }
     100%{
         opacity: 1;
         margin-top: 500px;
     }
 }

  2. 调用动画

  • animation-name:动画的时间
  • animation-duration:动画的完成时间
  • animation-delay:动画的延迟时间
  • animation-timing-function:动画的运动曲线
    • linear:均匀运动
    • ease:先慢后快再慢
    • ease-in:由慢开始
    • ease-in-out:由慢开始，由慢结束
    • ease-out:由慢结束
  • animation-iteration-count:动画的调运次数;
    • 可以写固定值，比如1，2 ，infinite(无限循环)
  • animation-direction: 动画的运动方向;
    • alternate-reverse:交替运动
  • animation-fill-mode:动画最终停止的位置
    • forwards：停在末时状态；backwards：停在初始状态

简写方式anmation: go 3s 0s linear ....; 注意:时间的参数位置第一个为动画完成时间，第二个为延迟时间

事件选择器过渡属性

  1. 指定变化的属性名称
  2. 动画的完成时间
  3. 动画的延迟时间
  4. 动画的运动曲线
    控制事件选择器的的动画效果
    transition: all 2s 0s linear;

    • src:音频存放的地址
